ChangeLog
---------

I **do not** intend to maintain another file by hand, so every section
of this ChangeLog is autogenerated by the following command:

    git log \
        --no-merges \
        --date=format:'%Y-%m-%d' \
        --pretty=format:'* %cd %H %s' \
        $(git ls-remote -t .|tail -1|cut -f1)..
